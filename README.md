# HR Onboarding & Employee Database System

A comprehensive Google Apps Script system for managing employee onboarding with document generation and checklist orchestration.

## Features

- **Employee Database**: Complete employee records with documents
- **Onboarding Checklists**: Automated task lists for new hires
- **Document Generation**: Auto-generate offer letters, contracts
- **Progress Tracking**: Track onboarding completion status
- **Role-Based Access**: HR Admin, Manager, Employee views

## Setup

1. Create Google Sheet with: Users, Employees, OnboardingChecklists, Documents, AuditLog
2. Deploy as Web App
